# 我的观点 | 个人博客网站

用深度视角，解读复杂世界。

## 访问地址

GitHub Pages: `https://mumfordragg5-jpg.github.io/my_website/`

## 技术栈

纯静态网站，无需构建工具：
- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts (Noto Sans SC)
- GitHub Pages 托管

## 功能

- 响应式设计，适配桌面和移动端
- 深色/浅色主题切换
- 文章阅读进度条
- 滚动加载动画

## 如何添加新文章

1. 在 `articles/` 目录下创建新的 HTML 文件
2. 在 `index.html` 的 `.article-grid` 中添加对应的文章卡片
