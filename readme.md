# 🏇 科技马前卒 | 国际财经深度解读

帮中国人读懂西方财经头条。

聚焦 Bloomberg / WSJ / NYT / The Economist，用深度视角解读全球财经大事。

## 访问地址

https://mumfordragg5-jpg.github.io/my_website/

## 功能

- 文章标签分类筛选（市场 / 热点 / 中国 / 地缘 / 能源）
- 文章搜索
- 深色 / 浅色主题切换
- 文章阅读进度条
- 响应式设计（桌面 + 移动端）
- GitHub Pages 自动部署

## 如何添加新文章

1. 在 `articles/` 下创建新 HTML 文件（可参考已有文章结构）
2. 在 `images/` 下添加对应封面图
3. 在 `index.html` 的 `article-grid` 中添加文章卡片
4. 推送到 `main` 分支，自动部署
